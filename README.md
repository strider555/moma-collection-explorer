# MoMA Collection Explorer

Interactive network graph visualization of [The Museum of Modern Art (MoMA)](https://www.moma.org/) collection, exploring relationships between classifications, mediums, and artists across 160,000+ artworks.

🔗 **Live demo**: https://strider555.github.io/moma-collection-explorer/

## Data

Collection metadata sourced from [MoMA's open collection dataset](https://github.com/MuseumofModernArt/collection), released under **CC0 1.0 Universal** (Public Domain Dedication).
